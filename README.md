Scrabble-Game
=============

Component-Based Programming .NET


Description:
- Design and implement a distributed multi-player game application that incorporates multiple assemblies and WCF.
- This game can be played with 1-4 players
- The game will play until the tile bag is empty

Purpose:
- Design a distributed application that reasonably conforms to the principles of distributed programming
- Implement, host, configure and consume a WCF service in a LAN computing environment
- Set-up and use callbacks on a client application from a WCF service

How to play multiplayer
- One player must have the server running
- On the same network another player can join by right clicking "TileClient" project > debug > start new instance.
- You should be able to see player two in your lobby.
- When start game is clicked both players will enter the game.
